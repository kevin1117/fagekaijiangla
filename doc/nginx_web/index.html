• Paxos 协议讲解

    Paxos 算法解决的问题是一个分布式系统如何就某个值（决议）达成一致。 一个典型的场景是，在一个分布式数据库系统中，如果各节点的初始状态一致， 每个节点执行相同的操作序列，那么他们最后能得到一个一致的状态。
